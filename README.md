# 🏠 Amazon Sales Analysis Dashboard (Power BI)

## 📌 Project Overview

This project analyzes **Amazon product sales performance** using an interactive **Power BI dashboard**.  
The dashboard highlights key business metrics such as **profit, orders, returns, product performance, and category distribution**.

The goal of this project is to demonstrate **data analysis, business intelligence, and data visualization skills** using Power BI.

## 🗃️ Business Problem

E-commerce companies sell thousands of products across multiple categories.  
Without proper analytics it becomes difficult to understand:

- Which products generate the most profit
- Which categories drive the most orders
- Which products are underperforming
- Where the sales are coming from geographically

This dashboard provides **clear insights to support data-driven decision making**.

---

## 📋 Dataset

The dataset contains **Amazon sales data** including:

- Product Name
- Category
- Subcategory
- Order Quantity
- Unit Cost
- Unit Price
- Revenue
- Profit
- Returns
- Sales Region

The data was cleaned and transformed using **Power Query** before visualization.

## 🔬 Key Performance Indicators (KPIs)

The dashboard tracks important business metrics:

- **Total Profit:** 1.98M
- **Total Orders:** 52
- **Total Returns:** 4

These KPIs help quickly evaluate **overall sales performance**.

---

## 📊 Dashboard Features

### 1. Category Distribution

Displays the total number of orders by product category:

- Accessories
- Bikes
- Clothing

This helps identify **high performing product segments**.

---

### 2. Sub-Category Order Analysis

A bar chart showing order distribution across product subcategories such as:

- Tires and Tubes
- Helmets
- Bottles and Cages
- Road Bikes
- Mountain Bikes
- Jerseys

This helps determine **which specific product groups drive demand**.

---

### 3. Product Profit / Loss Analysis

A table visualizing **profit contribution by individual products**.

Example products include:

- All-Purpose Bike Stand
- AWC Logo Cap
- Bike Wash – Dissolver
- Mountain Tire Tube
- Mountain-200 Series

This analysis helps identify **highly profitable products and low-performing products**.

---

### 4. Top Performing Products

Highlights the **top 3 profit-generating products**, helping businesses focus on their best-selling items.

Example:

- All-Purpose Bike Stand
- AWC Logo Cap
- Bike Wash – Dissolver

---

### 5. Global Sales Distribution

A map visualization shows **sales distribution across different geographic regions**, helping identify key markets.

---

## 💡 Key Insights

- **Accessories generate the highest number of orders**
- **Tires and Tubes is the most demanded subcategory**
- A small number of products contribute to **most of the total profit**
- **Return rates are low**, indicating stable product performance
